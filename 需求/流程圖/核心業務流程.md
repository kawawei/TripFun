# 核心業務流程圖

> **說明**: 本文件使用 Mermaid 語法繪製 TripFun App 的核心操作流程。

## 1. 核心導覽流 (Main Navigation)

```mermaid
flowchart TD
    Splash["啟動頁"] --> Home["行程列表 首頁"]
    Home -- "點擊加號" --> CreateTrip["建立新行程"]
    Home -- "點擊行程卡片" --> TripDetail["行程詳情 時間軸"]
    
    TripDetail -- "切換分頁" --> Timeline["每日時間軸"]
    TripDetail -- "切換分頁" --> Booking["預訂資訊庫"]
    TripDetail -- "切換分頁" --> Toolbox["工具箱"]
    
    Toolbox --> Translation["離線翻譯"]
    Toolbox --> Accounting["旅途記帳"]
    Toolbox --> Timezone["跨時區鏡"]
```

---

## 2. 行程編排流程 (Itinerary Planning Flow)

```mermaid
flowchart TD
    Start(["開始編排"]) --> SelectDate["選擇日期"]
    SelectDate --> InputActivity["輸入活動名稱與時間"]
    InputActivity --> ConflictCheck{"時間是否衝突?"}
    ConflictCheck -- "Yes" --> Alert["顯示紅色警示"]
    ConflictCheck -- "No" --> Save["儲存至時間軸"]
    Alert --> Save 
    Save --> End(["完成"])
```

---

## 3. 離線翻譯流程 (Offline Translation Flow)

```mermaid
flowchart TD
    Input["輸入待翻譯文字"] --> LangCheck{"目標語系包已下載?"}
    LangCheck -- "No" --> Download["提示並下載語系包"]
    Download --> Translate
    LangCheck -- "Yes" --> Translate["執行離線翻譯"]
    Translate --> Result["顯示翻譯結果"]
    Result --> FullScreen["點擊放大展示"]
```

---

## 4. 旅途記帳流程 (Expense Tracking Flow)

```mermaid
flowchart TD
    Entry["開啟記帳"] --> InputAmount["輸入金額"]
    InputAmount --> SelectCat["選擇分類"]
    SelectCat --> AutoRate{"是否有網路?"}
    AutoRate -- "Yes" --> FetchRate["抓取即時匯率"]
    AutoRate -- "No" --> LastRate["使用最後緩存匯率"]
    FetchRate --> Calc["換算台幣估值"]
    LastRate --> Calc
    Calc --> FinalSave["儲存記帳紀錄"]
```

---

## 5. 家屬協作共享流程 (Family Collaboration Flow)

```mermaid
flowchart TD
    SelectTrip["進入行程詳情"] --> ClickShare["點擊分享按鈕"]
    ClickShare --> GenToken["生成 UUID 安全 Token"]
    GenToken --> DeepLink["生成 Deep Link / QR Code"]
    DeepLink --> Send["發送給家人 (Line/WhatsApp)"]
    Send --> FamilyClick["家人點擊連結/掃碼"]
    FamilyClick --> JoinTrip{"加入行程?"}
    JoinTrip -- "Yes" --> CheckRole["指派為檢視者 (預設)"]
    CheckRole --> Sync["同步雲端行程資料"]
    Sync --> Finish(["協作開始"])
```
