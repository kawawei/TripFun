### 18.5 熱修復 (Hotfix)：終端機與開發者工具日誌洗版問題 [00:01]
- [x] **關閉 Dio `LogInterceptor` Response 攔截器**：
    - 問題原因：稍早導入 `StreamProvider` 搭配每 3 秒自動輪詢的功能後，因全域注入了 `LogInterceptor(responseBody: true)`，導致 Flutter 會每 3 秒印出整個 API 包含長文介紹的 JSON 回傳龐大陣列，造成終端機與瀏覽器開發者工具 (Console) 被瘋狂洗版甚至引發卡頓崩潰錯覺。
    - 解決方案：已將 `dio_provider.dart` 中的 `requestBody`、`responseBody` 與 `Headers` 監聽皆設定為 `false`，保持終端機暢通無阻。
